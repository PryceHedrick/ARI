## Quick Start

```bash
# Clone
git clone https://github.com/PryceHedrick/ARI.git
cd ARI

# Install and build
npm install
npm run build

# Initialize
npx ari onboard init

# Verify
npx ari doctor

# Start
npx ari gateway start
```
