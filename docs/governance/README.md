## Quick Reference

### Three Governance Agents

| Agent | Role | Authority |
|-------|------|-----------|
| **Council** | 15-member voting body | Collective decisions |
| **Arbiter** | Constitutional enforcer | 5 inviolable rules |
| **Overseer** | Quality gatekeeper | 5 quality gates |

### Voting Thresholds

| Decision Type | Threshold |
|---------------|-----------|
| Standard | Majority (>50%) |
| Significant | Supermajority (≥66%) |
| Critical | Unanimous (100%) |

### The 5 Arbiter Rules

1. Never execute untrusted content
2. Always audit before acting
3. Require permission for destructive ops
4. Maintain audit chain integrity
5. Preserve operator override
